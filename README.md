# -Coronary-Heart-Disease-Prediction
This project contains a machine learning pipeline that predicts the 10-year risk of Coronary Heart Disease (CHD) using the Framingham dataset. Built with Python and Scikit-Learn, it utilizes Logistic Regression to analyze patient metrics like age, cholesterol, and blood pressure, achieving a robust 83.39% accuracy score and a 0.72 ROC-AUC.
